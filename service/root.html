EIDAWS WFCatalog

The EIDAWS WFCatalog provides access to metadata and quality parameters for archives contributing data to EIDA. The supported request types are HTTP-GET and HTTP-POST. Service response is in JSON. Please refer to http://www.orfeus-eu.org/data/eida/eidaws/wfcatalog/ for a complete description of the service.

Available methods for WFCatalog

  • /query - interface to the WFCatalog
  • /version - dictates service version number
  • /application.wadl - WADL description of service

Feature Notes

  • Metadata extraction: ObsPy MSEEDMetadata
  • Database store: MongoDB
  • Service layer: NodeJS
